# Cluster jerárquico

En esta sección estudiaremos los modelos de cluster jerárquicos. Comenzamos cargando los datos que usaremos.

import pandas as pd
from sklearn import preprocessing

data_protein = pd.read_csv('https://raw.githubusercontent.com/jlaria/bigdata4/master/data/protein.csv', index_col=0)

scaler = preprocessing.StandardScaler().fit(data_protein)
df_scaled = scaler.transform(data_protein)

# AgglomerativeClustering

Para ajustar un modelo jerárquico en python podemos utilizar la clase AglomerativeClustering de sklearn.cluster

from sklearn.cluster import AgglomerativeClustering

model = AgglomerativeClustering(n_clusters=3, linkage='ward').fit(df_scaled)

La propiedad .labels_ contiene la agrupación obtenida a partir de este modelo de cluster jerárquico. En general cualquier algoritmo de agrupación no supervisada, puede verse de forma abstracta como una función que asigna etiquetas a datos, donde las etiquetas corresponden al índice de cada grupo final.

model.labels_

       0, 0, 2])

# Dendograma

Es posible visualizar el dendograma de un clúster construido con AgglomerativeClustering pero debemos crearnos nuestra propia función.

from matplotlib import pyplot as plt
from scipy.cluster.hierarchy import dendrogram
import numpy as np

def plot_dendrogram(model, **kwargs):
    # Create linkage matrix and then plot the dendrogram

    # create the counts of samples under each node
    counts = np.zeros(model.children_.shape[0])
    n_samples = len(model.labels_)
    for i, merge in enumerate(model.children_):
        current_count = 0
        for child_idx in merge:
            if child_idx < n_samples:
                current_count += 1  # leaf node
            else:
                current_count += counts[child_idx - n_samples]
        counts[i] = current_count

    linkage_matrix = np.column_stack([model.children_, model.distances_,
                                      counts]).astype(float)

    # Plot the corresponding dendrogram
    dendrogram(linkage_matrix, **kwargs)
    
model = AgglomerativeClustering(distance_threshold=0, n_clusters=None)

model = model.fit(df_scaled)
plt.title('Hierarchical Clustering Dendrogram')
# plot the top three levels of the dendrogram
plot_dendrogram(model, truncate_mode='level', p=3)
plt.xlabel("Number of points in node (or index of point if no parenthesis).")
plt.show()

Crear el dendograma, es mucho más sencillo con scipy.cluster, y podemos añadir labels fácilmente.

from scipy.cluster import hierarchy
from matplotlib import pyplot as plt

Z = hierarchy.linkage(df_scaled, 'ward')
plt.figure(figsize=(10,5))
dn = hierarchy.dendrogram(Z, labels=data_protein.index)

Además, podemos especificar dónde queremos cortar el árbol con el parámetro color_threshold.

plt.figure(figsize=(10,5))
dn = hierarchy.dendrogram(Z, labels=data_protein.index, color_threshold=5)

Otra opción es utilizar plotly para visualizar, con scipy.cluster para calcular las distancias.

import plotly.figure_factory as ff


fig = ff.create_dendrogram(df_scaled, 
                           labels=data_protein.index, 
                           color_threshold=5, 
                           linkagefun=lambda x: hierarchy.linkage(x, 'ward'))

fig.update_layout(width=700, height=500)
fig.show()

# Función linkage

Las funciones linkage más comunes están disponibles tanto en sklearn como en scipy. Ya hemos visto ward, pero también podemos escoger single, complete, average, weighted, centroid o median.

for linkage in ['single', 'complete', 'average', 'weighted', 'centroid', 'median']:
    fig = ff.create_dendrogram(df_scaled, 
                           labels=data_protein.index, 
                           linkagefun=lambda x: hierarchy.linkage(x, linkage))
    fig.update_layout(width=600, height=400, title=f'Linkage: {linkage}')
    fig.show()
